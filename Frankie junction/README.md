# 🌯 Frankie Junction

Premium Frankie shop website — responsive, animated, WhatsApp order system ke saath.

## 📁 Folder Structure

```
FrankieJunction/
├── index.html          ← Main website file
├── css/
│   └── style.css       ← All styling + animations
├── js/
│   └── app.js          ← Cart, orders, combo modal, star rating
├── images/             ← Apni photos yahan daalo
│   ├── logo.jpg        ← Navbar circle logo (200×200 recommended)
│   ├── hero.jpg        ← Hero section main image
│   ├── about.jpg       ← About section image
│   ├── paneer-tikka.jpg
│   ├── veg.jpg
│   ├── schezwan.jpg
│   ├── aloo.jpg
│   ├── corn-cheese.jpg
│   └── cheese-burst.jpg
└── README.md
```

## 🚀 How to Use

1. `index.html` ko browser mein open karo
2. `images/` folder mein apni real photos daalo (naam upar likhe hain)
3. `js/app.js` line 3 mein WhatsApp number already set hai: `916353598802`

## 📱 Features

- Sticky navbar with logo circle + animated brand name
- Hero section with floating image circle + gradient animation
- Menu cards with cart system (add/remove quantity)
- Combo offers with name/phone/location form → WhatsApp order
- Feedback section with star rating → WhatsApp send
- Scroll-reveal animations on all sections
- Counter animation on stats
- Floating WhatsApp button
- Fully responsive (mobile + tablet + desktop)
